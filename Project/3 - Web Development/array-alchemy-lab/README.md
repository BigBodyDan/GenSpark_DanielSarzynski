# Array Alchemy Lab
This project is a Node.js script that transforms a dataset of magic store items using JavaScript array methods.

### Features Implemented:
- Dataset with 15+ items, each with name, category, price, and power. 
- Prints all items using `forEach`.
- Filters items by price and category.
- Maps items to names and formatted strings.
- Generates a report including:
  - Total item count
  - Count of expensive items
  - Average price
  - Item with the highest power

### How to Run:
1. Make sure Node.js is installed.
2. Open a terminal in the project folder.
3. Run: `node array-alchemy.js`
