# Array Arena
This is a Node.js script that manages an item price roster using arrays.

### Features Implemented:
- Random array creation and updates
- Uses `push`, `pop`, `shift`, `unshift`, `includes`, `sort`
- Loop with 5 rounds of modifications
- Stats board with total, sum, average, min, max, and count > 50

### How to Run:
1. Make sure Node.js is installed.
2. Open a terminal in the project folder.
3. Run: `node array-arena.js`
