# The ES6 Spellbook
This project is a Node.js script that stores and formats spells using modern JavaScript features.

### Features Implemented:
- Array of 8+ spell objects with name, element, power, and mana cost. 
- Uses `const` by default and `let` only when needed. 
- Multiple arrow functions for formatting, calculations, and summaries. 
- Template literals for clean spell output. 
- Object and array destructuring in several places. 
- Spell Summary showing total spells, average power, and strongest spell.

### How to Run:
1. Make sure Node.js is installed.
2. Open a terminal in the project folder.
3. Run: `node spellbook.js`
