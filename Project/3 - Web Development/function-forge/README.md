# Function Forge
This project is a Node.js script that is a toolkit made of reusable functions.
It includes utilities to calculate tax totals, check even numbers,
convert text to title case, find a maximum value, format usernames,
and calculate averages.

#### The toolkit includes:
- 6 utility functions
- 1 validation function 
- 1 dispatcher/controller
- Function composition

### How to Run:
1. Make sure Node.js is installed.
2. Open a terminal in the project folder.
3. Run: `node function-forge.js`
